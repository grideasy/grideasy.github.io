GridEasy

Introduction

The idea behind GridEasy is a simple to use, but limited, introduction to trying out a grid design system for the first time. It provides a method of producing layouts that will appear differently depending on the viewing screen dimensions. Check out the examples section by altering the browser window size or viewing on a laptop, tablet or smarphone. GridEasy will allow you to produce single webpages with basic text content and images. If the idea of fluid grid design appeals to you and you want a more extensive design system checkout the commercial products mentioned in the above sites.

Content First Approach

Use you favourite text editor to write your content. Any new line with no tab set will be taken to be a heading, with a single tab a sub-heading and two tabs a paragraph. Three or more newlines between paragraphs will result in a new container for any following lines. Containers can then be edited in terms of size, position and format.

Layout First Approach

Breakpoints can be added for upto eight different states for various screen sizes. Grid dimesions can be set for each state. New containers can be created within any state which will then appear in all states. Text can be edited within any container and an image added if required. Whilst the text, image, position and format of a container remain consistent across all states their size is adjustable within each state.
