Cloner System

The cloner system allows you to build linear, radial, cuboid(matrix) or random constructions from a given set of one or more meshes, which themselves can have been made from the cloner system. It is also possible to clone meshes around the edges of an existing mesh using an objectCloner.

Each of the following playground examples are built from just one or two starter meshes.
